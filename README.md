## 📝 Description

GenAI_NotebookLMClone is a comprehensive web application designed to replicate the core functionalities of Google's NotebookLM, providing users with an intelligent environment for document analysis and research. Built with the Express.js framework, this project enables seamless interaction with generative AI to summarize complex content, answer contextual questions, and organize information efficiently. Developed as part of an SST GenAI assignment, it showcases the integration of modern web architecture with advanced AI-driven features to enhance the overall note-taking and research experience.
